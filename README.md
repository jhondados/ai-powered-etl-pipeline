## 🤖 AI Capabilities
- **Schema mapper**: maps source → target fields using LLM semantic matching (94% accuracy)
- **Type inference**: detects dates, currencies, CPF, CNPJ, phone formats automatically
- **Error self-healing**: when transformation fails, LLM generates fix and retries
- **Anomaly-aware**: pauses pipeline and alerts if data distribution shifts > 15%
- **Documentation**: auto-generates data dictionary and lineage docs
